## 삭제되는 데이터

계정 삭제 요청이 처리되면 아래 데이터가 **모두 영구 삭제**됩니다.

| 데이터 | 설명 |
|---|---|
| 계정 정보 | 카카오 계정 식별자, 닉네임, 이메일 주소 |
| 가족 정보 | 가족 이름, 초대코드, 구성원 정보 |
| 아이 정보 | 아이 이름, 색상 설정 |
| 일정 정보 | 등하원 활동, 요일, 시간, 장소, 알림 설정 |
| 예외 정보 | 특정 날짜의 시간 변경·건너뛰기 기록 |
| 기기 정보 | 푸시 알림 토큰 |

> 가족의 마지막 관리자가 계정을 삭제하면, 해당 가족의 모든 데이터(아이·일정 포함)가 함께 삭제됩니다.

## 보관되는 데이터

- 삭제 요청 처리 후 **별도로 보관하는 개인정보는 없습니다.**
- 서버 운영 기록(접속 로그 등)은 개인을 식별할 수 없는 형태로 **최대 30일간** 보관된 후 자동 삭제됩니다.

---

## 계정을 삭제하지 않고 일부 데이터만 삭제하기

계정을 유지한 채 일부 데이터만 지우고 싶다면, 앱 내에서 직접 삭제할 수 있습니다.

- **일정 삭제**: 홈 화면에서 활동을 누른 뒤 `활동 삭제`
- **아이 삭제**: `설정 → 아이 관리`에서 해당 아이의 `삭제`
  (아이를 삭제하면 그 아이의 모든 일정도 함께 삭제됩니다)
